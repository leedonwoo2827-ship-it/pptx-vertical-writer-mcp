# 템플릿 가이드 (내부 문서)

사업자가 MD 작성 시 "어떤 형태로 글을 써야 하는지" 참고하는 가이드입니다.

---

## T0. 구분페이지

**용도**: 섹션/챕터 구분, 목차 시작 페이지

**필요한 글**: 제목 1줄

**예시**:
```
@content_1: 1.2 추진전략
```

---

## T1. 카드형 다중 (가장 많이 사용)

**용도**: 현황/문제점/개선방향, 접근전략/수행체계, 과제 분석

**필요한 글**: 카드 2~6개 (각 카드 = 제목 + 내용)

**글 형태**:
- 카드 제목: 15자 이내, 핵심 키워드
- 카드 내용: 2~4문장, 300자 이내
- 상단 요약: 1~2문장 (governing_message)

**예시**:
```
@governing_message: 1차 사업의 성과와 한계를 분석하여 2차 사업의 방향을 설정함
@카드1_제목: 제도적 과제
@카드1_내용: UBEC 중심 ESP 추진 중이나, 스마트스쿨 운영의 법적·제도적 기반이 체계화되지 않음. 21개교 확대 시 표준운영절차(SOP) 수립 필요.
@카드2_제목: 기술적 과제
@카드2_내용: 21개교 통합관리 시스템(IDC, LMS, 대시보드) 부재. 전력·인터넷 인프라의 지역별 격차.
```

---

## T2. 카드+다이어그램

**용도**: 사업 목적, 전략 개요 (장기/단기 목적 + 세분화)

**필요한 글**: 카드 2~3개 + 세분화 불릿 + 정책적 의의

**예시**:
```
@카드1_제목: 장기목적
@카드1_내용: 나이지리아 전역 교육환경 개선 및 교육역량 강화
@카드2_제목: 단기목적  
@카드2_내용: 21개 스마트스쿨 교육 품질 제고
@content_1: (장기목적 세분화 불릿 4개)
@content_2: (단기목적 세분화 불릿 4개)
@section_title_1: 스마트교육 확산과 품질 제고
```

---

## T3. 범위/개요 (거버닝메시지)

**용도**: 사업 범위, 핵심 개요, 비전 선언

**필요한 글**: 거버닝메시지(핵심 1~2문장) + 본문 설명 + 영역 그리드

**예시**:
```
@governing_message: 본 사업의 범위는 6개 권역 21개 스마트스쿨을 대상으로 5대 Output을 통합 추진하는 PMC 사업입니다.
@content_1: (핵심 문구)
@content_2: (상세 설명, 3~5문장)
```

---

## T6. 순수 데이터테이블

**용도**: 큰 데이터 표, 지표, 상세 스펙, 일정

**필요한 글**: 마크다운 테이블

**예시**:
```
@governing_message: 사업 산출물별 주요 내용

| 구분 | 산출물 | 주요 내용 |
|---|---|---|
| Output 1.1 | 디지털 학습 환경 | 스튜디오 구축 |
| Output 1.2 | 통합 시스템 | LMS, 대시보드 |
```

---

## T7. 테이블+설명shape (프로세스)

**용도**: 프로세스 흐름, 단계별 활동 + 데이터

**필요한 글**: 단계 설명 + 테이블

**예시**:
```
@governing_message: 성과관리 7단계 활동
@heading_1: 통합성과관리(IPM)
@content_1: RBM 방법론 기반 7단계 활동 수행

| 단계 | 활동 | 시기 |
|---|---|---|
| 1 | 성과관리 워크숍 | 착수 시 |
| 2 | 핵심 지표 합의 | 1분기 |
```

---

## T9. 핵심메시지/다이어그램

**용도**: 핵심 포인트 나열, CSF, 성공요소, 원칙, 방향성

**필요한 글**: 핵심 문구 3~6개 (각 50~100자)

**예시**:
```
@governing_message: 5대 핵심성공요소(CSF) 도출
@breadcrumb: 1-1. 사업이해도 > 1-1-7. CSF
@content_1: CSF 1: 1차 사업 성과의 체계적 계승 및 확장
@content_2: CSF 2: 데이터 기반 의사결정 및 성과관리
@content_3: CSF 3: 현지 주도의 자립적 운영체계 구축
@content_4: CSF 4: 교육과정 정합성 확보 및 현장 적용성 극대화
@content_5: CSF 5: 정책적 지속가능성과 제도적 연계
```

---

## 공통 규칙

1. **@governing_message**: 모든 슬라이드 상단에 표시되는 핵심 메시지. 200자 이내.
2. **@breadcrumb**: 섹션 경로 (예: "1-1. 사업이해도 > 1-1-5. 현황")
3. **카드 내용**: 한 카드당 300자 이내
4. **색상 마커**: `{{red:rawdata 출처}}`, `{{green:references 출처}}`
5. **테이블**: 표준 마크다운 테이블 형식
